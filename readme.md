# SandwichJS expression parser

Javascript AST parser for 'simple' expressions.

## Supported expressions

* binary operators (+-%/)

## Roadmap

### Add support for

* a || b || 'thiees'
* (a * 2) + 10
* funcA() || funcB()
* [1, 2]
* {foo: 'bar'}
* !!foo
* ('foo')
* [['foo'], [2]]
* true ? 'yes' : 'no'
* true
* false
* null
